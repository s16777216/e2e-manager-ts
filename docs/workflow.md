1. 使用者參數:

```json
{
  "id": "login_success",
  "name": "登入流程",
  "steps": [
    "進入 Login 頁面",
    "輸入帳號",
    "輸入密碼",
    "點擊登入按鈕",
    "跳轉到首頁"
  ],
  "expected": "跳轉到首頁, 並顯示登入成功訊息"
}
```

2. 伺服器接受到請求
3. 輸入步驟(step) n, 使用 tools, 直到完成步驟 n
4. 截圖結果並存檔
5. 執行下一個步驟 n + 1 (回到第 3 步)
6. 完成所有步驟, LLM 根據截圖結果判斷是否成功
7. 測試案例結束
